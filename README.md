#  Luminescence Data Analyser

A Python-based data science toolkit for analysing 
photoluminescence (PL) emission spectra of 
lanthanide-doped nanophosphors.

##  Features
- Temperature-dependent PL spectral analysis
- Peak detection using SciPy
- Thermal quenching behaviour
- Publication-quality visualisations

## 🛠️ Technologies
Python | Pandas | NumPy | Matplotlib | SciPy | Jupyter

##  Results
- Peak emission at 480nm across all temperatures
- Classic thermal quenching confirmed
- Intensity reduces ~21% from 300K to 400K

## Project Structure
- `01_spectral_analysis.ipynb` → Main analysis notebook
- `data/` → Spectral dataset (CSV)
- `outputs/` → Generated plots
